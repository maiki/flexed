    @mixin triangle() { ... }

    Description

    Mixin for creating triangles. This mixin creates triangles.

    Parameters

    None.

    Example

    Example

    .foo {
   @include triangle(down, 1em, #4fade3);
}

    Output

    .foo {
     top: 100%;
     left: 1em;
     position: absolute;
     width: 0;
     height: 0;
     content: '';
     z-index: 2;
     border-top: 1.5em solid #3498db;
     border-left: 1em solid transparent;
     border-right: 1em solid transparent;
 }

    breakpoint

    @mixin breakpoint($name) { ... }

    Description

    Media query Mixin for map based breakpoint. This mixin creates a media query block

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $name

    thin(30em) OR normal(48em) OR wide(61.25em) OR full(80em)

    		Name—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Example

    @include breakpoint(full) {
  .baz { color: black; }
}

    Output

    @media (min-width: 60em) {
  .baz { color: black; }
}

    Requires

    • [variable] breakpoints
